# 💝 情侣纪念网页使用说明

> 一个纯前端实现的浪漫纪念网页，完美适配手机和电脑端

---

## 📁 项目文件结构

```
项目文件夹/
├── index.html          # 主页面文件
├── style.css           # 样式文件
├── script.js           # 交互脚本
├── 使用说明.md         # 本说明文档
├── images/             # 照片文件夹
│   ├── cover.jpg       # 封面照片（必须）
│   ├── photo1.jpg      # 第1页照片
│   ├── photo2.jpg      # 第2页照片
│   ├── ...
│   ├── photo15.jpg     # 第15页照片
│   ├── heart.gif       # 爱心动图（可选）
│   ├── star.gif        # 星光动图（可选）
│   └── petal.gif       # 花瓣动图（可选）
└── audio/              # 音频文件夹
    └── bg-music.mp3    # 背景音乐文件
```

## 🚀 快速开始

### 方法1：本地查看
1. 双击打开 `index.html` 文件
2. 网页会在默认浏览器中打开
3. 点击"点击开启回忆"按钮开始浏览

### 方法2：部署到网络（推荐）
部署后可通过网址分享给TA：

#### GitHub Pages（免费推荐）
1. 注册 GitHub 账号（github.com）
2. 创建新仓库（Repository）
3. 上传所有文件到仓库
4. 在仓库 Settings → Pages 中启用
5. 获得网址：`https://你的用户名.github.io/仓库名/`

#### Vercel（最简单）
1. 访问 vercel.com 注册
2. 点击 "New Project"
3. 导入文件夹或连接 GitHub
4. 自动部署，获得网址

#### Netlify
1. 访问 netlify.com
2. 直接拖拽项目文件夹到页面
3. 自动部署完成

## ✏️ 内容修改指南

### 📷 第一步：准备照片

#### 需要的照片文件：
1. **封面照片**（1张）：命名为 `cover.jpg`
2. **内页照片**（15张）：命名为 `photo1.jpg` 到 `photo15.jpg`
3. **装饰动图**（可选3张）：`heart.gif`、`star.gif`、`petal.gif`

#### 照片要求：
- 格式：JPG、PNG、WEBP 均可
- 尺寸建议：800×600 像素以上
- 大小建议：每张不超过 2MB

#### 操作步骤：
1. 将照片重命名为对应的文件名
2. 复制到 `images` 文件夹中
3. 完成！无需修改代码

> 💡 **重要提示**：文件名必须完全一致（包括大小写和扩展名）

---

### ✍️ 第二步：修改文字

打开 `index.html` 文件，找到标注区域：

```html
<!-- 【文本替换区 - 第1页】修改下方引号内的文字 -->
<p class="love-text">"在这里写您的情话"</p>
```

#### 修改位置清单：
- **封面页标题**：第34行附近
- **第1页文字**：第52行附近
- **第2页文字**：第66行附近
- **第3页文字**：第80行附近
- ...以此类推

所有需要修改的位置都有 `【文本替换区】` 标注。

#### 文字修改建议：
- 每段情话建议控制在50字以内
- 保持引号不变
- 可以添加emoji表情 ❤️ 💕 🌹

---

### 🎵 第三步：添加背景音乐

#### 准备音乐文件：
1. 选择一首您喜欢的音乐
2. 转换为 MP3 格式（推荐）
3. 重命名为 `bg-music.mp3`
4. 放入 `audio` 文件夹

#### 音频要求：
- 格式：MP3（推荐）、WAV、OGG
- 大小：建议不超过 10MB
- 时长：3-5分钟为佳（会自动循环）

#### 音乐来源建议：
- 网易云音乐下载
- QQ音乐下载
- 免费音乐网站（如 Freesound）

> ⚠️ **注意**：某些浏览器会阻止音乐自动播放，需要用户点击页面后才能播放

---

### 🎨 第四步：修改装饰动图（可选）

如果您想要自定义飘落的爱心、星光、花瓣等装饰效果：

1. 准备GIF动图文件
2. 重命名为：`heart.gif`、`star.gif`、`petal.gif`
3. 放入 `images` 文件夹
4. 建议使用透明背景的GIF

如果不想要装饰效果，可以删除对应文件。

## 🎯 功能说明

### 📱 响应式设计
- ✅ 手机端：照片占屏幕80%宽度
- ✅ 电脑端：照片占屏幕50%宽度
- ✅ 平板端：自动适配

### 🎮 交互方式

#### 电脑端：
- **鼠标点击**：点击"下一页"按钮
- **键盘快捷键**：
  - `→` 或 `Enter`：下一页
  - `←`：上一页
  - `Home` 或 `Esc`：返回首页
  - `空格`：播放/暂停音乐

#### 手机端：
- **触摸滑动**：
  - 向左滑动：下一页
  - 向右滑动：上一页
- **点击按钮**：导航翻页
- **点击右上角按钮**：控制音乐

### 🎨 视觉效果
- ✅ 照片淡入动画（1秒）
- ✅ 文字渐现效果
- ✅ 按钮脉动动画
- ✅ 页面切换过渡（0.8秒）
- ✅ 浪漫背景装饰（柔和飘动）

## ❓ 常见问题解答

### Q1: 为什么照片显示不出来？
**A:** 请检查：
1. 照片文件名是否正确（区分大小写）
2. 照片是否放在 `images` 文件夹中
3. 照片格式是否支持（jpg、png、webp）
4. 文件路径是否正确

### Q2: 为什么音乐不播放？
**A:** 可能原因：
1. 浏览器阻止了自动播放（点击页面任意位置即可播放）
2. 音乐文件名不正确（必须是 `bg-music.mp3`）
3. 音乐文件损坏（尝试更换文件）
4. 点击右上角音乐按钮手动播放

### Q3: 如何修改照片数量？
**A:** 如果需要增加或减少页数：
1. 在 `index.html` 中复制/删除整个页面区块
2. 修改 `script.js` 中的 `totalPages` 变量
3. 准备相应数量的照片文件

### Q4: 可以在手机上直接修改吗？
**A:** 可以，推荐使用：
- 手机端代码编辑器（如 QuickEdit、Acode）
- 在线代码编辑器（如 CodeSandbox）
- GitHub 网页版直接编辑

### Q5: 如何更换字体？
**A:** 在 `style.css` 第8行修改：
```css
font-family: 'your-font', Georgia, serif;
```
