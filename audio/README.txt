🎵 音频文件夹使用说明
═══════════════════════════════════════════════════════════

【重要】请将您的背景音乐文件放入此文件夹

✅ 必需文件：
────────────────────────────────────
bg-music.mp3    - 背景音乐文件

文件名必须完全一致（包括大小写和扩展名）

═══════════════════════════════════════════════════════════
📝 音频要求
═══════════════════════════════════════════════════════════

✅ 文件格式：MP3（推荐）、WAV、OGG
✅ 文件大小：建议不超过 10MB
✅ 音乐时长：3-5分钟为佳（会自动循环播放）
✅ 音质要求：128kbps 以上

═══════════════════════════════════════════════════════════
🔄 替换音乐步骤
═══════════════════════════════════════════════════════════

方法1（推荐）：
1. 选择您喜欢的音乐文件
2. 重命名为：bg-music.mp3
3. 复制到本文件夹中
4. 如提示覆盖，选择"是"

方法2：
1. 先删除文件夹中的示例音乐
2. 将您的音乐文件复制进来
3. 重命名为：bg-music.mp3

❓ 常见问题
═══════════════════════════════════════════════════════════

Q: 为什么音乐不播放？
A: 1. 检查文件名是否为 bg-music.mp3
   2. 点击页面任意位置（浏览器限制自动播放）
   3. 点击右上角音乐按钮手动播放

Q: 可以使用其他格式的音频吗？
A: 可以，但需要修改 HTML 代码中的文件扩展名

Q: 音乐太大怎么办？
A: 使用音频压缩工具降低比特率（128kbps 即可）

Q: 可以添加多首歌曲吗？
A: 可以，需要修改 JavaScript 代码实现播放列表

Q: 手机上音乐为什么不自动播放？
A: 大多数手机浏览器禁止音频自动播放，需要用户交互后才能播放
